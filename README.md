# Weather-app
A sleek and user-friendly weather app built with HTML, CSS, and JavaScript with API. Just input a city, and instantly get real-time weather details including temperature, humidity, UV index, etc and comprehensive air quality parameters like O3, NO2, and SO2, PM2.5 etc, basically an "Advanced Weather-app".

# Features
- Live Weather Data: Get up-to-date weather information for your city worldwide.
- Real Time Temperature: Gives you current temperature in degree celcius.
- Additional Info: Like Humidity, Dewpoint, Visibility, Wind Speed, UV Index.
- AQI Parameters: Like CO, No2, O3, PM2.5, PM10.

# Technologies Used
- Frontend: HTML, CSS, JAVASCRIPT
- API: weatherapi.com
- Icons: Fontawesome
